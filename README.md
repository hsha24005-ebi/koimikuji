# 恋みくじ（最終決戦版）

ボタンをクリックするだけで、今日の恋愛運を占うことができるシンプルなウェブアプリケーションです。

## 特徴

- **シンプル操作**: 「おみくじを引く」ボタンを押すだけで即座に結果が表示されます。
- **ランダム表示**: JavaScriptを用いて、4種類の運勢（大吉、中吉、小吉、凶）をランダムに抽出します。
- **和風モダンなデザイン**: 茶色と淡いベージュ（ブラッシュドアーモンド）を基調とした、落ち着いたデザインを採用しています。

## ファイル構成

プロジェクトは以下の3つのファイルで構成されています。

- `index.html`: アプリの構造（タイトル、ボタン、表示エリア）を定義。
- `style.css`: 背景色、フォント、ボタンのホバーエフェクトなどのデザインを定義。
- `script.js`: おみくじのロジック（ランダムな運勢の生成と表示）を制御。

## 使い方

1. 全てのファイル（`index.html`, `style.css`, `script.js`）を同じフォルダに保存します。
2. ブラウザで `index.html` を開きます。
3. 「おみくじを引く」ボタンをクリックして、運勢を確認してください。

## 運勢の内容

おみくじの結果には以下の4つが含まれています。

- **大吉💖**: 素敵な出会いがあるかも！
- **中吉😊**: 少し勇気を出すと良いことが起きる
- **小吉😐**: 今日はゆっくり過ごそう
- **凶😥**: 無理せず慎重に行動しよう

## カスタマイズについて

運勢の種類を増やしたい場合は、`script.js` 内の `fortunes` 配列にメッセージを追加するだけで、自動的に抽選対象に含まれるようになります。
